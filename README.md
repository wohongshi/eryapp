# ERY - 二次元游戏卡池展示

横屏展示多个二次元游戏的版本角色卡池信息和主线任务。

## 支持的游戏及版本数

| 游戏 | 图标 | 版本范围 | 版本数 |
|------|------|----------|--------|
| 原神 | ⚡ | 1.0 ~ 月之八 | 51个 |
| 崩坏：星穹铁道 | ⭐ | 1.0 ~ 4.4 | 30个 |
| 绝区零 | 🎯 | 1.0 ~ 3.0 | 19个 |
| 鸣潮 | 🌊 | 1.0 ~ 3.5 | 20个 |
| 异环 | 🔮 | 1.0 ~ 1.2 | 3个 |
| 明日方舟：终末地 | 🛡️ | 1.0 ~ 1.4 | 5个 |

## 功能特性

- 📱 横屏全屏显示
- 🎮 左侧游戏选择栏
- 📅 版本时间线选择
- 📖 主线任务名称展示
- 📝 版本剧情梗概
- 👥 角色卡池展示
- 🆕 新角色标记
- ⭐ 稀有度显示
- 🎨 深色主题

## 技术栈

- Flutter 3.x
- Material Design 3
- 横屏锁定

## 构建

```bash
# 获取依赖
flutter pub get

# 构建APK
flutter build apk --release
```

## 项目结构

```
lib/
├── main.dart              # 入口
├── data/
│   └── game_repository.dart  # 游戏数据
├── models/
│   └── game_data.dart     # 数据模型
└── screens/
    ├── home_screen.dart   # 主页
    └── game_detail_screen.dart  # 游戏详情
```
